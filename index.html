Descargar McBedrock

💻 McBedrock Para PC 💻

Paso 1: Descarga el Runtime de Windows

Es necesario instalar este complemento para que el launcher funcione correctamente.

  • windowsdesktop-runtime-8.0.11-win-x86.exe (32 bits)
  • windowsdesktop-runtime-8.0.11-win-x64.exe (64 bits)

Si no sabes si tu PC es de 32 o 64 bits, busca **"Acerca de tu PC"** en el menú de inicio y mira la sección **"Tipo de sistema"**.

Paso 2: Descarga el BedrockLauncher

Una vez instalado el Runtime, descarga el launcher y ejecútalo.

BedrockLauncher 2025.8.25.24.rar

📱 McBedrock Para Teléfono 📱

Descarga la versión para tu dispositivo

  • minecraft_1_21_101(compressed_32).apk (lite)
